Conclusion

I've chosen these colors because I wanted to try out a more nuanced palette that potentially could be applied to an app, or have been used in an informative website. By using the 60-30-10 rule for this color palette, the light/white/blue had to convey a relaxing air,
but at the same time I’d like to achieve a nice flow with the stronger turquoise, which leads the eye further down the page.
Using the complimentary color to turpoise/green as the accent,
the call to action buttons for example, gets a nice contrast to the primary and secondary colors and highlights the CTA’s visibility.
Converting the color palette into a dark mode I opted for toning down the saturation and hue for the existing color palette.
